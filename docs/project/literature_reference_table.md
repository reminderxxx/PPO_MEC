1. 先查重：用标题、DOI、arXiv ID 或官方页面链接在本文件中搜索；同一论文的 preprint、early access 和正式版本只保留一行，优先使用正式 DOI 或会议/期刊官网链接。

## 写作 Claim 模板

可采用的谨慎表述：

- Existing VEC studies have investigated service caching, task offloading, migration, and mixed-timescale decisions, but they usually optimize these dimensions separately or for single-shot task/service abstractions.
- Recent LoRA and LLM serving systems show that adapters, checkpoints, and KV caches are first-class scheduling states; however, their setting is mostly data-center serving, not mobility-driven RSU handoff.
- PPO_MEC targets the intersection: continuous DAG workflow execution across RSUs, AI adapter cache cooperation, handoff state migration, and multi-controller RL under a real NGSIM + Alibaba data chain.

不应采用的过度表述：
